# Modulor Massing

A browser-based parametric architectural massing tool — preset typologies,
punched window facades, and seeded randomization, built with Three.js.

**Live:** [modulor-massing.axisbim.io](https://modulor-massing.axisbim.io)

Generates massing studies from a proportional module derived from Le
Corbusier's Modulor Blue Series, with three synced views (3D, Plan,
Elevation), four color palettes, and Simple/Advanced control modes.
Single-file, no build step.

More detail on presets, invariants, and project structure: see
[`EXTENDED.md`](./EXTENDED.md).
